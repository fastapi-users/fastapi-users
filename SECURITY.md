# Security Policy

## Supported Versions

We provide support for the latest version and recommend to upgrade as soon as a new version is released.

## Reporting a Vulnerability

If you have identified a vulnerability, please report it using [GitHub Security Advisory](https://github.com/fastapi-users/fastapi-users/security/advisories/new) tool, to ensure confidentiality and security.

We'll review it as soon as possible and publish a fix accordingly.
